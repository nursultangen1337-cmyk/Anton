# iOS Shortcut: Аналізатор тестів

Автоматичний аналіз тестових завдань: сфоткав → отримав відповідь у нотифікації.

## Що потрібно

- iPhone з iOS 16 або новіше
- API ключ Anthropic (Claude)

## Як отримати API ключ

1. Зайди на [console.anthropic.com](https://console.anthropic.com)
2. Зареєструйся або увійди
3. Перейди в **API Keys**
4. Натисни **Create Key**
5. Скопіюй ключ (починається з `sk-ant-...`) — він показується тільки один раз!
6. Поповни баланс (мінімум $5). Один запит коштує ~$0.01-0.03

## Створення Shortcut — покроково

Відкрий додаток **Shortcuts** (Команди) на iPhone.

### Крок 1: Новий Shortcut

1. Натисни **+** у верхньому правому куті
2. Натисни на назву зверху → перейменуй на **"Аналіз тесту"**

### Крок 2: Take Photo (Зробити фото)

1. Натисни **Add Action** (Додати дію)
2. Знайди **"Take Photo"** (Зробити фото)
3. Додай цю дію
4. Залиш **"Show Camera Preview"** увімкненим, щоб бачити що фоткаєш

### Крок 3: Resize Image (Змінити розмір)

1. Натисни **+** під попередньою дією
2. Знайди **"Resize Image"** (Змінити розмір зображення)
3. Встанови ширину: **1024**
4. Auto Height: увімкнено

### Крок 4: Base64 Encode (Кодування)

1. Натисни **+**
2. Знайди **"Base64 Encode"** (Base64-кодування)
3. Переконайся що вхід (input) — це **Resized Image** з попереднього кроку

### Крок 5: Text (JSON запит)

1. Натисни **+**
2. Знайди **"Text"** (Текст)
3. Встав цей текст:

```
{"model":"claude-sonnet-4-20250514","max_tokens":256,"messages":[{"role":"user","content":[{"type":"image","source":{"type":"base64","media_type":"image/jpeg","data":"BASE64_PLACEHOLDER"}},{"type":"text","text":"Це фото тестового завдання українською мовою. Уважно прочитай питання та всі варіанти відповідей. Вкажи правильну відповідь та дай коротке пояснення (1-2 речення). Відповідай українською."}]}]}
```

4. **ВАЖЛИВО:** У тексті знайди слово `BASE64_PLACEHOLDER` і видали його
5. Натисни на те місце → з'явиться меню змінних → вибери **Base64 Encoded** (результат попереднього кроку)

### Крок 6: Get Contents of URL (HTTP запит)

1. Натисни **+**
2. Знайди **"Get Contents of URL"** (Отримати вміст URL)
3. URL: `https://api.anthropic.com/v1/messages`
4. Натисни **"Show More"** і налаштуй:
   - **Method**: `POST`
   - **Headers** — додай 3 заголовки:
     - `x-api-key` : `ТВІЙ_API_КЛЮЧ` (заміни на свій ключ sk-ant-...)
     - `anthropic-version` : `2023-06-01`
     - `content-type` : `application/json`
   - **Request Body**: вибери **File**
   - Як файл вкажи **Text** з попереднього кроку

### Крок 7: Get Dictionary Value (Дістати відповідь)

Тут потрібно 3 дії одна за одною:

**7a.**
1. Натисни **+**
2. Знайди **"Get Dictionary Value"** (Отримати значення зі словника)
3. Key: `content`
4. Input: **Contents of URL** (результат попереднього кроку)

**7b.**
1. Натисни **+**
2. Знайди **"Get Item from List"** (Отримати елемент зі списку)
3. Вибери **First Item** (Перший елемент)
4. Input: результат кроку 7a

**7c.**
1. Натисни **+**
2. Знайди **"Get Dictionary Value"** (Отримати значення зі словника)
3. Key: `text`
4. Input: результат кроку 7b

### Крок 8: Show Notification (Push-нотифікація)

1. Натисни **+**
2. Знайди **"Show Notification"** (Показати сповіщення)
3. Title: `Відповідь`
4. Body: вибери результат кроку 7c (змінна **Dictionary Value**)

## Готово! Як користуватися

Тепер Shortcut працює так:
1. Відкриваєш Shortcuts → натискаєш **"Аналіз тесту"**
2. Відкривається камера → фоткаєш тест
3. Через 3-5 секунд приходить нотифікація з відповіддю

## Швидкий запуск

### Іконка на Home Screen
1. У Shortcuts натисни **...** на своєму Shortcut
2. Натисни іконку налаштувань зверху (стрілка вниз або **i**)
3. **"Add to Home Screen"** (Додати на головний екран)
4. Вибери іконку та натисни **Add**

Тепер одна іконка на робочому столі — натиснув і одразу камера.

### Back Tap (подвійний тап по задній кришці)
1. **Settings** → **Accessibility** → **Touch** → **Back Tap**
2. Вибери **Double Tap** або **Triple Tap**
3. Знайди **"Аналіз тесту"** в списку Shortcuts
4. Готово!

Тепер просто постукай 2 рази по задній кришці iPhone → камера → відповідь.

## Вирішення проблем

| Проблема | Рішення |
|----------|---------|
| Помилка 401 | Неправильний API ключ. Перевір `x-api-key` в кроці 6 |
| Помилка 400 | Неправильний JSON. Перевір що `BASE64_PLACEHOLDER` замінено на змінну Base64 Encoded |
| Довго чекати | Перевір інтернет. Claude відповідає за 3-10 секунд |
| Нотифікація обрізана | Натисни на нотифікацію щоб побачити повний текст |
| Помилка "Could not run" | Перевір що всі кроки з'єднані правильно (кожен отримує input з попереднього) |
| Недостатньо коштів | Поповни баланс на console.anthropic.com |
